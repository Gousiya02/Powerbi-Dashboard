# Financial Dashboard using Power BI

## Overview

This project presents an interactive **Power BI dashboard** built using a financial dataset. 
The dashboard provides insights into key business metrics such as **Sales, Profit, and Cost of Goods Sold (COGS)**,
helping analyze performance across different products, countries, and time periods.

## Objectives

* Analyze overall financial performance
* Track sales and profit trends over time
* Compare product and country-wise performance
* Visualize KPIs for better decision-making
  
##  Dataset

* Source: Financial dataset (Excel/CSV)
* Key fields:
  * Sales
  * Profit
  * COGS
  * Product
  * Country
  * Date/Month/Year

## Process & Steps

### 1. Data Import

* Loaded dataset into Power BI from Excel/CSV

### 2. Data Cleaning & Transformation

* Handled missing values
* Ensured correct data types (date, numeric, text)
* Used **Power Query Editor** for transformations

### 3. Data Modeling

* Structured data for analysis
* Created calculated fields where required

### 4. DAX Measures

Created key metrics using DAX:
* Total Sales
* Total Profit
* Total COGS
* Sales vs Target

##  Dashboard Features

* **KPI Cards**
  
  * Total Sales
  * Total Profit
  * Total COGS

* **Visualizations**

  * Bar Chart → Profit by Product
  * Donut Chart → Profit by Country
  * Line Chart → Sales Trend by Month
  * Area Chart → Profit Trend
  * Stacked Chart → Profit by Country & Product
  * Gauge Chart → Sales vs Target

* **Filters**

  * Year slicer for dynamic analysis

## Key Insights

* Highest profit generated by top-performing products
* Sales peak observed during specific months
* Certain countries contribute more significantly to overall profit

##  Tools & Technologies

* Power BI
* Microsoft Excel / CSV

##  Conclusion

This project demonstrates how Power BI can be used to transform raw financial data into meaningful insights through interactive visualizations and KPIs.
